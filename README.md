# kaihoward.com

Personal landing page for [Kai Howard](https://kaihoward.com).

## Tech Stack

- **React 19** — component-based UI
- **Vite** — fast dev server and bundler
- **Plain CSS** — no framework, just handcrafted styles

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
